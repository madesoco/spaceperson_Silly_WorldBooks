# 冒险者等级是衡量冒险者综合实力信誉与过往功绩的评定体系。它不仅决定了冒险者能承接的委托难度，也直接影响其社会地位与收入水平。
ranks:
  - level: F
    title: 见习者
    description: "刚注册的新人，是冒险者生态系统的基石。他们的生活围绕着城镇内的基础委托展开，目标是积累经验与最基本的生存资金，为更换一套过得去的装备而努力。每一次完成委托都是一次信心的建立。"
    representative:
      name: 菲莉娅·菲尔温
      summary: "一位充满活力的翼人少女，每日在小镇"巨人之踵"处理着镇民们的各种杂务，从寻找走失的宠物到采集悬崖上的草药，享受着帮助他人的充实生活。"

  - level: E
    title: 摸索者
    description: "初出茅庐的冒险者，已经能够处理城镇周边的低威胁任务。他们开始尝试组建固定的小队，学习如何与同伴协同作战，生活质量略有改善，偶尔能在酒馆享用一顿烤肉与麦酒。"

  - level: D
    title: 合格者
    description: "具备独立处理常规威胁能力的合格冒险者，是协会的中坚力量。他们是守护商路清剿哥布林哨站的主力，摆脱了新手的稚嫩，开始有了作为冒险者的自觉与自信。"

  - level: C
    title: 资深者
    description: "经验丰富的冒险者，通常是某个领域小有名气的专家。他们有能力领导小队执行较为复杂的任务，如探索古代遗迹的浅层区域，或是在小规模的怪物潮中保护村庄。"

  - level: B
    title: 精英
    description: "在某个地区或领域内享有盛誉的精英冒险者，其实力足以影响局部战局。B级以上的冒险者会被地方领主奉为座上宾，接受的委托往往直接来自贵族或领主，例如讨伐地行龙或调查威胁整个地区的神秘事件。"

  - level: A
    title: 英雄
    description: "活着的传奇，其实力与功绩足以与国家级的英雄媲美，是大师阶的典范。他们的名字在王国间流传，每一次行动都备受关注，其决定能够改变历史的走向。"
    representative:
      name: 阿塔兰忒
      identifier: 风行者
      summary: "一位以速度与精确箭术闻名的精灵游侠，是现役A级冒险者中的佼佼者，无数新人冒险者憧憬的目标。"

  - level: S
    title: 神话
    description: "规格之外的存在，他们的力量与事迹已经达到了神话的领域。此等级并非通过常规晋升获得，而是授予那些完成了足以被载入史册的伟业，拥有匹敌神话生物甚至天灾般力量的怪物。他们的出现本身就意味着将有动摇世界根基的重大事件发生。"
